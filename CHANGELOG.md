### Features
- **Public Access**: Home, About, Departments pages viewable by all
- **Member Features**: Dashboard, department joining, photo gallery access
- **Admin Features**: Complete management of departments, members, announcements, and photos
- **Security**: Protected routes, input validation, secure file uploads
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

### Technical Stack
- Node.js with Express.js
- MongoDB with Mongoose ODM
- EJS templating engine
- Bootstrap 5 with Font Awesome icons
- Multer for file uploads
- bcrypt for password hashing
- Session-based authentication
